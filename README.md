# Travel Journal

<p align="center">
  <img src="screenshots/Home_TravelJournal.png" width="350">
</p>

A cross-platform mobile application built with Flutter that helps travelers plan future trips, organize travel experiences, and preserve memories from previous journeys.

## Overview

Travel Journal combines trip planning, travel inspiration, and personal travel history into a single mobile application.

Unlike traditional itinerary apps, the project focuses on both planning future travel and documenting completed experiences.

The application was developed using Flutter and Dart as a university software engineering capstone project.

## Features

### Trip Planning

- Create travel itineraries
- Build customizable packing lists
- Save destinations for future trips
- Organize travel ideas and bucket lists

### Travel Journal

- Track visited continents, countries, regions, and cities
- Record famous landmarks and points of interest
- Save restaurants and local cuisine
- Document events and experiences
- Keep notes and travel memories

### Personal Travel Statistics

- Countries visited
- Continents explored
- Cities discovered
- Foods tried
- Events attended
- Friends traveled with

### Media & Memories

- Store travel photos
- Save travel notes
- Keep restaurant recommendations

## Motivation

Many travel applications focus on booking flights or creating itineraries.

This project was designed to combine planning, inspiration, and memory keeping into a single mobile experience.
